# 🎮 Sliding Puzzle Game

The Sliding Puzzle Game is a web-based interactive game built using HTML, CSS, and JavaScript. The objective of the game is to arrange shuffled numbered tiles in the correct order by sliding them into the empty space.

## 🚀 Features

- Dynamic tile generation
- Shuffle functionality
- Click-based tile movement
- Win condition detection
- Responsive user interface

## 🛠️ Technologies Used

- HTML – Structure of the game
- CSS – Styling and layout design
- JavaScript – Game logic and DOM manipulation

## 💡 How to Play

1. Click on a tile next to the empty space.
2. Slide tiles to arrange them in the correct order.
3. Complete the puzzle to win the game.

## 📌 Learning Outcome

This project helped me improve my JavaScript fundamentals, DOM manipulation skills, and problem-solving ability by building an interactive web application without using external libraries.
